# Kilangi-jewellery-frontend
The project recreates the provided desktop and mobile UI designs with close attention to layout, typography, spacing, colors, and responsiveness. It is fully responsive across screen sizes and includes subtle JavaScript interactions to enhance user experience while keeping the design clean and minimal.

Tech Stack
- HTML5 – Semantic structure  
- CSS3 – Flexbox, Grid, Media Queries  
- JavaScript (Vanilla JS) – Basic interactions  


Features
- Fully responsive design (Desktop & Mobile)
- Pixel-close UI implementation based on provided PDFs
- Clean and modular code structure
- Subtle hover and interaction effects using JavaScript
- Optimized layout for different screen sizes
